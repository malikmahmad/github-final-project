# Simple Interest Calculator

This project contains a simple interest calculator implemented as a Bash script.

## Introduction

A simple interest calculator that computes interest based on user-provided inputs.

## Formula

Simple Interest = (P × R × T) / 100

Where:
- P = Principal amount
- R = Rate of interest (per annum)
- T = Time period (in years)

## Usage

Run the Bash script and provide the following inputs when prompted:
- Principal (P)
- Rate of interest (R)
- Time period (T)

The script will output the calculated Simple Interest value.

## Notes
Ensure bc is installed on your system to run the script.
